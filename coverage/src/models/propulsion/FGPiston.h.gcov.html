     115             : <h3>Definition of the piston engine configuration file parameters:</h3>
     116             : Basic parameters:
     117             : - \b minmp - this value is the nominal idle manifold pressure at sea-level
     118             :       without boost. Along with idlerpm, it determines throttle response slope.
     119             : - \b maxmp - this value is the nominal maximum manifold pressure at sea-level
     120             :       without boost. Along with maxrpm it determines the resistance of the
     121             :       aircraft intake system. Overridden by air-intake-impedance-factor
     122             : - \b man-press-lag - Delay in seconds for manifold pressure changes to take effect
     123             : - \b starter-torque - A value specifing the zero RPM torque in lb*ft the starter motor
     124             :       provides. Current default value is 40% of the horse power value.
     125             : - \b starter-rpm - A value specifing the maximum RPM the unloaded starter motor
     126             :       can achieve. Loads placed on the engine by the propeller and throttle will
     127             :       further limit RPM achieved in practice.
     128             : - \b idlerpm - this value affects the throttle fall off and the engine stops
     129             :       running if it is slowed below 80% of this value. The engine starts
     130             :       running when it reaches 80% of this value.
     131             : - \b maxrpm - this value is used to calculate air-box resistance and BSFC. It 
     132             :       also affects oil pressure among other things.
     133             : - \b maxhp - this value is the nominal power the engine creates at maxrpm. It
     134             :       will determine bsfc if that tag is not input. It also determines the
     135             :       starter motor power.
     136             : - \b displacement - this value is used to determine mass air and fuel flow
     137             :       which impacts engine power and cooling.
     138             : - \b cycles - Designate a 2 or 4 stroke engine. Currently only the 4 stroke
     139             :       engine is supported.
     140             : - \b bore - cylinder bore is currently unused.
     141             : - \b stroke - piston stroke is used to determine the mean piston speed. Longer
     142             :       strokes result in an engine that does not work as well at higher RPMs.
     143             : - \b compression-ratio - the compression ratio affects the change in volumetric
     144             :       efficiency with altitude.
     145             : - \b sparkfaildrop - this is the percentage drop in horsepower for single
     146             :       magneto operation.
     147             : - \b static-friction - this value is the power required to turn an engine that 
     148             :       is not running. Used to control and slow a windmilling propeller. Choose
     149             :       a small percentage of maxhp.
     150             : 
     151             : Advanced parameters
     152             : - \b bsfc - Indicated Specific Fuel Consumption. The power produced per unit of
     153             :       fuel. Higher numbers give worse fuel economy. This number may need to be
     154             :       lowered slightly from actual BSFC numbers because some internal engine 
     155             :       losses are modeled separately. Typically between 0.3 and 0.5
     156             : - \b volumetric-efficiency - the nominal volumetric efficiency of the engine.
     157             :       This is the primary way to control fuel flow  Boosted engines may require
     158             :       values above 1. Typical engines are 0.80 to 0.82
     159             : - \b air-intake-impedance-factor - this number is the pressure drop across the
     160             :       intake system. Increasing it reduces available manifold pressure. Also a 
     161             :       property for run-time adjustment.
     162             : - \b ram-air-factor - this number creates increases manifold pressure with an
     163             :       increase in dynamic pressure (aircraft speed).
     164             :       Also a property for run-time adjustment.
     165             : 
     166             : Cooling control:
     167             : - \b cylinders  - number of cylinders scales the cylinder head mass.
     168             : - \b cylinder-head-mass - the nominal mass of a cylinder head. A larger value
     169             :       slows changes in engine temperature
     170             : - \b cooling-factor - this number models the efficiency of the aircraft cooling
     171             :       system. Also a property for run-time adjustment.
     172             : 
     173             : Supercharge parameters:
     174             : - \b numboostspeed -  zero (or not present) for a naturally-aspirated engine,
     175             :       either 1, 2 or 3 for a boosted engine.  This corresponds to the number of
     176             :       supercharger speeds.  Merlin XII had 1 speed, Merlin 61 had 2, a late
     177             :       Griffon engine apparently had 3.  No known engine more than 3, although
     178             :       some German engines had continuously variable-speed superchargers.
     179             : - \b boostoverride - whether or not to clip output to the wastegate value
     180             : - \b boost-loss-factor - zero (or not present) for 'free' supercharging. A value entered
     181             :       will be used as a multiplier to the power required to compress the input air. Typical 
     182             :       value should be 1.15 to 1.20.
     183             : - \b boostmanual - whether a multispeed supercharger will manually or
     184             :       automatically shift boost speeds. On manual shifting the boost speeds is
     185             :       accomplished by controlling the property propulsion/engine/boostspeed.
     186             : - \b takeoffboost - boost in psi above sea level ambient. Typically used for
     187             :       takeoff, and emergency situations, generally for not more than five
     188             :       minutes.  This is a change in the boost control setting, not the actual
     189             :       supercharger speed, and so would only give extra power below the rated altitude.
     190             :       A typical takeoff boost for an early Merlin was about 12psi, compared
     191             :       with a rated boost of 9psi.
     192             : 
     193             :       When TAKEOFFBOOST is specified in the config file (and is above RATEDBOOST1), 
     194             :       the throttle position is interpreted as:
     195             :      - 0 to 0.98 : idle manifold pressure to rated boost (where attainable)
     196             :      - 0.99, 1.0 : takeoff boost (where attainable).
     197             : 
     198             : The next items are all appended with either 1, 2 or 3 depending on which
     199             : boostspeed they refer to:
     200             : - \b ratedboost[123] - the absolute rated boost above sea level ambient
     201             :       (14.7 PSI, 29.92 inHg) for a given boost speed.
     202             : 
     203             : - \b ratedpower[123] - unused
     204             : - \b ratedrpm[123] - The rpm at which rated boost is developed
     205             : - \b ratedaltitude[123] - The altitude up to which the rated boost can be
     206             :       maintained. Up to this altitude the boost is clipped to rated boost or
     207             :       takeoffboost. Beyond this altitude the manifold pressure must drop,
     208             :       since the supercharger is now at maximum unregulated output. The actual
     209             :       pressure multiplier of the supercharger system is calculated at
     210             :       initialization from this value.
     211             : 
     212             :     @author Jon S. Berndt (Engine framework code and framework-related mods)
     213             :     @author Dave Luff (engine operational code)
     214             :     @author David Megginson (initial porting and additional code)
     215             :     @author Ron Jensen (additional engine code)
     216             :     @see Taylor, Charles Fayette, "The Internal Combustion Engine in Theory and Practice"
     217             :   */
