      57             : /** Models a helicopter rotor.
      58             : 
      59             : 
      60             : <h3>Configuration File Format</h3>
      61             : @code
      62             : <rotor name="{string}">
      63             :   <diameter unit="{LENGTH}"> {number} </diameter>
      64             :   <numblades> {number} </numblades>
      65             :   <gearratio> {number} </gearratio>
      66             :   <nominalrpm> {number} </nominalrpm>
      67             :   <minrpm> {number} </minrpm>
      68             :   <maxrpm> {number} </maxrpm>
      69             :   <chord unit="{LENGTH}"> {number} </chord>
      70             :   <liftcurveslope Xunit="1/RAD"> {number} </liftcurveslope>
      71             :   <twist unit="{ANGLE}"> {number} </twist>
      72             :   <hingeoffset unit="{LENGTH}"> {number} </hingeoffset>
      73             :   <flappingmoment unit="{MOMENT}"> {number} </flappingmoment>
      74             :   <massmoment Xunit="SLUG*FT"> {number} </massmoment>
      75             :   <polarmoment unit="{MOMENT}"> {number} </polarmoment>
      76             :   <inflowlag> {number} </inflowlag>
      77             :   <tiplossfactor> {number} </tiplossfactor>
      78             :   <maxbrakepower unit="{POWER}"> {number} </maxbrakepower>
      79             :   <gearloss unit="{POWER}"> {number} </gearloss>
      80             :   <gearmoment unit="{MOMENT}"> {number} </gearmoment>
      81             : 
      82             :   <controlmap> {MAIN|TAIL|TANDEM} </controlmap>
      83             :   <ExternalRPM> {number} </ExternalRPM>
      84             : 
      85             :   <groundeffectexp> {number} </groundeffectexp>
      86             :   <groundeffectshift unit="{LENGTH}"> {number} </groundeffectshift>
      87             : 
      88             : </rotor>
      89             : 
      90             : //  LENGTH means any of the supported units, same for ANGLE and MOMENT.
      91             : //  Xunit-attributes are a hint for currently unsupported units, so
      92             : //  values must be provided accordingly.
      93             : 
      94             : @endcode
      95             : 
      96             : <h3>Configuration Parameters:</h3>
      97             : 
      98             :   Brief description and the symbol frequently found in the literature.
      99             : 
     100             : <pre>
     101             :     \<diameter>           - Rotor disk diameter (2x R).
     102             :     \<numblades>          - Number of blades (b).
     103             :     \<gearratio>          - Ratio of (engine rpm) / (rotor rpm), usually > 1.
     104             :     \<nominalrpm>         - RPM at which the rotor usally operates.
     105             :     \<minrpm>             - Lowest RPM used in the model, optional and defaults to 1.
     106             :     \<maxrpm>             - Largest RPM used in the model, optional and defaults to 2 x nominalrpm.
     107             :     \<chord>              - Blade chord, (c).
     108             :     \<liftcurveslope>     - Slope of curve of section lift against section angle of attack,
     109             :                              per rad (a).
     110             :     \<twist>              - Blade twist from root to tip, (theta_1).
     111             :     \<hingeoffset>        - Rotor flapping-hinge offset (e).
     112             :     \<flappingmoment>     - Flapping moment of inertia (I_b).
     113             :     \<massmoment>         - Blade mass moment. Mass of a single blade times the blade's
     114             :                              cg-distance from the hub, optional.
     115             :     \<polarmoment>        - Moment of inertia for the whole rotor disk, optional.
     116             :     \<inflowlag>          - Rotor inflow time constant, sec. Smaller values yield to quicker
     117             :                               responses (typical values for main rotor: 0.1 - 0.2 s).
     118             :     \<tiplossfactor>      - Tip-loss factor. The Blade fraction that produces lift.
     119             :                               Value usually ranges between 0.95 - 1.0, optional (B).
     120             : 
     121             :     \<maxbrakepower>      - Rotor brake, 20-30 hp should work for a mid size helicopter.
     122             :     \<gearloss>           - Friction in gear, 0.2% to 3% of the engine power, optional (see notes).
     123             :     \<gearmoment>         - Approximation for the moment of inertia of the gear (and engine),
     124             :                               defaults to 0.1 * polarmoment, optional.
     125             : 
     126             :     \<controlmap>         - Defines the control inputs used (see notes).
     127             : 
     128             :     \<ExternalRPM>        - Links the rotor to another rotor, or an user controllable property.
     129             : 
     130             :     Experimental properties
     131             : 
     132             :     \<groundeffectexp>    - Exponent for ground effect approximation. Values usually range from 0.04
     133             :                             for large rotors to 0.1 for smaller ones. As a rule of thumb the effect
     134             :                             vanishes at a height 2-3 times the rotor diameter.
     135             :                               formula used: exp ( - groundeffectexp * (height+groundeffectshift) )
     136             :                             Omitting or setting to 0.0 disables the effect calculation.
     137             :     \<groundeffectshift>  - Further adjustment of ground effect, approx. hub height or slightly above
     138             :                             (This lessens the influence of the ground effect).
     139             : 
     140             : </pre>
     141             : 
     142             : <h3>Notes:</h3>
     143             : 
     144             :   <h4>- Controls -</h4>
     145             : 
     146             :     The behavior of the rotor is controlled/influenced by following inputs.<ul>
     147             :       <li> The power provided by the engine. This is handled by the regular engine controls.</li>
     148             :       <li> The collective control input. This is read from the <tt>fdm</tt> property
     149             :            <tt>propulsion/engine[x]/collective-ctrl-rad</tt>. See below for tail rotor</li>
     150             :       <li> The lateral cyclic input. Read from
     151             :            <tt>propulsion/engine[x]/lateral-ctrl-rad</tt>.</li>
     152             :       <li> The longitudinal cyclic input. Read from 
     153             :            <tt>propulsion/engine[x]/longitudinal-ctrl-rad</tt>.</li>
     154             :       <li> The tail rotor collective (aka antitorque, aka pedal) control input. Read from
     155             :            <tt>propulsion/engine[x]/antitorque-ctrl-rad</tt> or
     156             :            <tt>propulsion/engine[x]/tail-collective-ctrl-rad</tt>.</li>
     157             : 
     158             :     </ul>
     159             : 
     160             :   <h4>- Tail/tandem rotor -</h4>
     161             : 
     162             :     Providing <tt>\<ExternalRPM\> 0 \</ExternalRPM\></tt> the tail rotor's RPM
     163             :     is linked to to the main (=first, =0) rotor, and specifing
     164             :     <tt>\<controlmap\> TAIL \</controlmap\></tt> tells this rotor to read the
     165             :     collective input from <tt>propulsion/engine[1]/antitorque-ctrl-rad</tt>
     166             :     (The TAIL-map ignores lateral and longitudinal input). The rotor needs to be
     167             :     attached to a dummy engine, e.g. an 1HP electrical engine.
     168             :     A tandem rotor is setup analogous. 
     169             : 
     170             :   <h4>- Sense -</h4>
     171             : 
     172             :     The 'sense' parameter from the thruster is interpreted as follows, sense=1 means
     173             :     counter clockwise rotation of the main rotor, as viewed from above. This is as a far
     174             :     as I know more popular than clockwise rotation, which is defined by setting sense to
     175             :     -1. Concerning coaxial designs - by setting 'sense' to zero, a Kamov-style rotor is
     176             :     modeled (i.e. the rotor produces no torque).
     177             : 
     178             :   <h4>- Engine issues -</h4>
     179             : 
     180             :     In order to keep the rotor/engine speed constant, use of a RPM-Governor system is
     181             :     encouraged (see examples).
     182             : 
     183             :     In case the model requires the manual use of a clutch the <tt>\<gearloss\></tt>
     184             :     property might need attention.<ul>
     185             : 
     186             :     <li> Electrical: here the gear-loss should be rather large to keep the engine
     187             :          controllable when the clutch is open (although full throttle might still make it
     188             :          spin away).</li>
     189             :     <li> Piston: this engine model already has some internal friction loss and also
     190             :          looses power if it spins too high. Here the gear-loss could be set to 0.25%
     191             :          of the engine power (which is also the approximated default).</li>
     192             :     <li> Turboprop: Here the default value might be a bit too small. Also it's advisable
     193             :          to adjust the power table for rpm values that are far beyond the nominal value.</li>
     194             : 
     195             :     </ul>
     196             : 
     197             :   <h4>- Scaling the ground effect -</h4>
     198             : 
     199             :     The property <tt>propulsion/engine[x]/groundeffect-scale-norm</tt> allows fdm based
     200             :     scaling of the ground effect influence. For instance the effect vanishes at speeds
     201             :     above approx. 50kts, or one likes to land on a 'perforated' helipad.
     202             : 
     203             :   <h4>- Development hints -</h4>
     204             : 
     205             :     Setting <tt>\<ExternalRPM> -1 \</ExternalRPM></tt> the rotor's RPM is controlled  by
     206             :     the <tt>propulsion/engine[x]/x-rpm-dict</tt> property. This feature can be useful
     207             :     when developing a FDM.
     208             :
