     100             : </propeller>
     101             : ~~~
     102             : 
     103             : ### Configuration Parameters
     104             : 
     105             : <pre>
     106             :     \<ixx>           - Propeller rotational inertia.
     107             :     \<diameter>      - Propeller disk diameter.
     108             :     \<numblades>     - Number of blades.
     109             :     \<gearratio>     - Ratio of (engine rpm) / (prop rpm).
     110             :     \<minpitch>      - Minimum blade pitch angle.
     111             :     \<maxpitch>      - Maximum blade pitch angle.
     112             :     \<minrpm>        - Minimum rpm target for constant speed propeller.
     113             :     \<maxrpm>        - Maximum rpm target for constant speed propeller.
     114             :     \<constspeed>    - 1 = constant speed mode, 0 = manual pitch mode. 
     115             :     \<reversepitch>  - Blade pitch angle for reverse.
     116             :     \<sense>         - Direction of rotation (1=clockwise as viewed from cockpit,
     117             :                         -1=anti-clockwise as viewed from cockpit). Sense is
     118             :                        specified in the parent tag of the propeller.
     119             :     \<p_factor>      - P factor. It is specified in the parent tag of
     120             :                        the propeller.
     121             :     \<ct_factor>     - A multiplier for the coefficients of thrust.
     122             :     \<cp_factor>     - A multiplier for the coefficients of power.
     123             : </pre>
     124             : 
     125             : Two tables are needed. One for coefficient of thrust (Ct) and one for
     126             : coefficient of power (Cp).
     127             : 
     128             : Two tables are optional. They apply a factor to Ct and Cp based on the
     129             : helical tip Mach.
     130             : 
     131             : The parameters <sense> and <p_factor> must be specified at the parent level i.e.
     132             : in the <thruster> element. This allows to specify different sense and P factor
     133             : values for each propeller of the model while using the same definition file for
     134             : all the propellers.
     135             : 
     136             : In addition to thrust, the propeller applies two moments to the aircraft:
     137             : - The torque that tends to roll the aircraft in the direction opposite to the
     138             : propeller rotation,
     139             : - and the gyroscopic moment.
     140             : 
     141             : It should be noted that historically the gyroscopic moment had an incorrect
     142             : sign. The correct sign can be obtained by specifying a **version** attribute
     143             : higher than 1.0 to the propeller definition
     144             : ~~~.xml
     145             : <propeller name="a_prop" version="1.1">
     146             :   <!-- propeller definition -->
     147             : </propeller>
     148             : ~~~
     149             : For backward compatibility, the absence of the **version** attribute will result
     150             : in the gyroscopic moment to be computed with the legacy incorrect sign.
     151             : 
     152             : Several references were helpful, here:
     153             : + Barnes W. McCormick, "Aerodynamics, Aeronautics, and Flight Mechanics",
     154             : Wiley & Sons, 1979 ISBN 0-471-03032-5
     155             : + Edwin Hartman, David Biermann, "The Aerodynamic Characteristics of
     156             : Full Scale Propellers Having 2, 3, and 4 Blades of Clark Y and R.A.F. 6
     157             : Airfoil Sections", NACA Report TN-640, 1938 (?)
     158             : + Various NACA Technical Notes and Reports
     159             : 
     160             : @author Jon S. Berndt
     161             : @see FGEngine
     162             : @see FGThruster
